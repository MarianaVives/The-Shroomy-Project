# The-Shroomy-Project
Repositorio para proyecto informativo de los hongos

Este proyecto se usa para crear un sitio informativo acerca de los distintos tipos de hongos silvestres y sus posibles usos y efectos en nuestra salud.

La informacion utilizada para descripcion de los hongos es obtenida a traves de GPT por lo que se recomienda leer con discrecion. 

No recomiendo el uso de esta pagina web para ser utilizada como guia para la recoleccion de hongos silvestres.
